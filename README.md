# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, certifications, and achievements. The portfolio highlights my work in Data Structures & Algorithms, Data Science, dashboards, and web development. It serves as a central place where recruiters and collaborators can explore my technical abilities and projects.

## Features

* About Me section with my background and career interests
* Skills section highlighting my technical expertise
* Projects showcasing my work and practical implementations
* Certifications and achievements
* Clean and responsive design

## Technologies Used

* HTML
* CSS
* JavaScript
* Git & GitHub for version control and deployment

## Purpose

The goal of this portfolio is to present my work, demonstrate my problem-solving abilities, and provide an easy way for others to learn about my technical journey and projects.
